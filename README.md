# Organizador de Arquivos Automático 📂🤖

Script desenvolvido em **Python** para automatizar a organização de pastas, movendo arquivos para diretórios específicos com base em suas extensões (PDFs, Imagens, Planilhas e Scripts).

## ✨ Funcionalidades
- Identificação automática de extensões de arquivos.
- Criação dinâmica de pastas caso elas não existam.
- Organização instantânea com um único clique.

## 🛠 Tecnologias
- **Python 3.14**
- Bibliotecas nativas: `os` e `shutil` (não precisa instalar nada!).

## 🚀 Como usar
1. Coloque o script `organizador.py` na pasta que deseja organizar.
2. Execute o comando:
   ```bash
   python organizador.py
